Installation

To view the template correctly in your browser when you are working locally, you must have to a Local Server application like XAMPP, or their respective versions for Windows (WAMP) and Mac(MAMP). This is because this template uses PHP, and this language need be executed by a server.

If you are working via cloud Hosting server, you shouldn't have any problem to view the template when you hosted it.

Basic configuration

Blat is amazing! Is super easy to customize. In two minutes, you be made the basic configuration to start to enjoy the template.

Open in your code editor the "config.php" file, and simply follow the comments.

In this file you will can configure:

  • The title of your website
  • The color scheme: Colors avaliables: red, gold and dark
  • Your favicon
  • Your logo
  • The email what you use to receive the forms messages
  • The Mailchimp action url
  • The coordinates where will be placed the event. (This tool can help you)
  • The color for your map
  • The Google Webfonts you will use
  • The Google Analytics tracking code
  • Add the Parallax Effect on any class of your website

Wow! Its super easy!

Customize your content

When you have done the basic configuration, you already can customize the content for your Landing Page.

You have three templates, all customizables: the index (index.php), the menu card page (menu.php), and another single page (single.php).

Customize the Menu:

You can personalize the menu with your items. The ID assigned in the url items allow to situate the section directly instead of scrolling down. It's important type the same items in Responsive Menu, so this menu will appear in screens to less than 780px of width.

Image Slideshow:

Add more slides adding more list items ("li"), or remove the slideshow to show a more clean and flat style for header of your Landing Page.

Change images and add your text:

Are avaliable to you "blockquote" tag and a ".important" class to hightlight a text. Images uploaded will adapt its size to show in webpae correctly.

Add, move, style or remove blocks:

8 different blocks are avaliables to you (one-column, two-columns, three-columns, four-columns, left-quote, right-quote, testimonials and restaurant_booking).

  • Colored: Add in any of these blocks the class "colored" to add a default colored background color.
  • Red, gold, dark, green, grey and blue: Add in any of these blocks the class "red", "gold", "dark", "green", "grey" or "blue" to add this background color in your block.
  • Background image: Add in any of these blocks the class "background" to add any background image in your block (customize this image in CSS).
  • Pattern: Add in any of these blocks the class "pattern" to add a pattern as background image in your block (customize this pattern in CSS).
  • Animations: Add in any of these blocks the class "slide-down", "slide-left", "slide-right" to add a nice animation effect in your block.
Testimonials slideshow:

Add more slides adding more list items ("li").
